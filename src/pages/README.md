# Pages

This directory contains the top-level Vue route pages for the desktop application.

## Files

- `Ai.vue` — runs single-job AI matching analysis with resume input and optional context.
- `AiReports.vue` — lists cached AI reports, supports filtering, paging, and report inspection.
- `Crawl.vue` — manages browser-assisted crawling, runtime controls, and crawl logs.
- `Export.vue` — exports stored job data in supported formats.
- `Jobs.vue` — browses, searches, and manages collected job records.
- `ResumeWorkspace.vue` — guides the resume workflow from source import and diagnosis through module confirmation, transient acceptance feedback, stale-final detection, and final resume generation/export.
- `Settings.vue` — edits application settings such as browser path and OpenAI-compatible API options.
