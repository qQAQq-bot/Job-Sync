# Resume Workspace Components

This directory contains focused Vue components used by the resume workspace feature.

## Files

- `ResumeBasicProfileForm.vue` — edits the candidate's structured profile fields.
- `ResumeCandidatePreview.vue` — shows generated candidate copy and provides accept/regenerate actions.
- `ResumeDiagnosisView.vue` — renders the AI diagnosis summary for each resume module.
- `ResumeFinalPreview.vue` — displays the assembled final resume, final actions, and the warning when confirmed module content has changed after final generation.
- `ResumeModuleEditor.vue` — collects per-module input and follow-up answers before candidate generation.
- `ResumeWorkspaceNav.vue` — renders workspace step navigation and completion state.
